-----------------------------GENESIS SOURCE CODE-------------------------



1) SOURCECODE folder includes all the files required for GENESIS 
   project (including database files) to run. 

2) You will be required to have Visual Basic 6.0 installed in order
   to open the project. 

3) For full project documentation and help please refer to DOCUMENTATION.doc.

4) You can copy the GENESIS_SOURCECODE folder to any place on your hard drive.
   Afterwards, open the SOURCECODE folder and run the Genesis.vbp file.
